# Mini ERP Web Application

## 🚀 Project Overview
A secure Mini ERP Web Application built using Python (Flask) and SQLite.  
The system features session-based authentication, inventory management, and supplier management.

## 🔐 Features
- User Login & Session Authentication
- Dashboard Interface
- Add & View Products
- Add & View Suppliers
- SQLite Database Integration
- Clean & Responsive UI using CSS

## 🛠 Technologies Used
- Python 3
- Flask
- SQLite
- HTML5 & CSS3
